# 利用snakemake搭建疾病分析流程

## 分析步骤如下所示 （全部选择开源第三方软件）
1. 质控：fastap
2. 比对：bwa
3. 变异检出：samtools or GATK 
4. 注释：annovar（annovar有些注释不符合hgvs规则）
5. 变异过滤（频率，功能区等）
6. 样本分析：家系分析 / 散发分析 / 单个位点权重分析
7. 通路分析
8. 疾病相关性分析
9. 疾病用药指导
